Tale of the Red Cliffs

Vivez l'expérience palpitante des batailles légendaires des Trois Royaumes dans le jeu Tale of the Red Cliffs.

Appuyez sur les boucliers dorés et turquoises pour découvrir des symboles identiques et gagner des prix fabuleux! Surveillez les flèches de feu; lorsqu'elles atteignent un symbole, un symbole multiplicateur de jade peut ressurgir des cendres!

Comment miser

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (–) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter votre solde du prix d’un billet et commencer la partie.

Comment jouer

Dans le jeu Tale of the Red Cliffs, vous gagnez lorsque certains de VOS SYMBOLES sont identiques aux SYMBOLES GAGNANTS.

Appuyez sur les boucliers dorés pour faire apparaître les SYMBOLES GAGNANTS. Appuyez sur les boucliers turquoises pour faire apparaître VOS SYMBOLES et les lots correspondants. Lorsque tous les symboles sont apparents, des flèches de feu peuvent atteindre VOTRE SYMBOLE et faire apparaître un multiplicateur de jade de 2, 5, 10 ou 20.

Si VOTRE SYMBOLE est identique à un SYMBOLE GAGNANT, vous gagnez le lot associé à VOTRE SYMBOLE.

Le lot associé à VOTRE SYMBOLE est multiplié lorsque vous obtenez un symbole multiplicateur de jade.

Le jeu prend fin lorsque tous les boucliers dorés et tous les boucliers turquoises ont affiché leurs symboles respectifs. Le jeu affiche un message indiquant le résultat de la partie.

Tout afficher

Appuyez sur TOUT AFFICHER pour que la partie se joue automatiquement. La partie continue jusqu’à ce que tous les symboles aient été dévoilés.

Pour continuer la partie en mode manuel, appuyez sur ARRÊTER.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les lots instantanés sont ajoutés aux gains totaux.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
